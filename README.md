# NumPy Student Marks Analysis

This is a beginner-level Python project that demonstrates how to use NumPy for analyzing student academic performance.

## Features
- Subject-wise average marks
- Student-wise total and average marks
- Grade assignment based on average marks
- Pass/Fail status using NumPy vectorized conditions

## Technologies Used
- Python
- NumPy

## Learning Outcomes
- Understanding NumPy arrays
- Using axis=0 and axis=1 correctly
- Applying conditional logic with np.where
- Performing data analysis without explicit loops
